# 💸 Convert - Conversor de Moedas

O **Convert** é uma aplicação web simples desenvolvida para converter valores monetários de diferentes moedas (como Dólar, Euro e Libra) para Real (BRL). Este projeto foi desenvolvido como parte de um desafio prático da **Rocketseat** para consolidação de conceitos fundamentais do JavaScript no ecossistema Front-End.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais:

- **HTML5:** Estruturação semântica da aplicação.
- **CSS3:** Estilização moderna, layout responsivo e design focado na experiência do usuário.
- **JavaScript:** Manipulação da DOM, tratamento de eventos, validação de inputs com expressões regulares (RegEx) e formatação de moedas com `Intl.NumberFormat`.

---

## 🛠️ Funcionalidades

- **Validação Inteligente:** O campo de entrada de valor aceita estritamente números, utilizando Regex para remover caracteres de texto em tempo real.
- **Conversão Dinâmica:** Seleção de múltiplas moedas (USD, EUR, GBP) para conversão instantânea em Reais (BRL).
- **Formatação de Moeda:** Exibição dos resultados formatados de forma profissional com base na localidade padrão do país utilizando a API nativa do JavaScript.
- **Tratamento de Erros:** Alertas e feedbacks visuais amigáveis caso ocorra algum imprevisto no processo de conversão.
